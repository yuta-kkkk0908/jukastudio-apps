# プライバシーポリシー

ForgeNote（以下「本サービス」）は、ユーザーのプライバシーを尊重し、以下のとおり情報の取り扱いを定めます。

## 1. 取得する情報
本サービスでは、ユーザーが入力した内容（タイトル、ジャンル、対象読者、ハッシュタグ等）を、プロンプト生成のために利用します。  
また、利便性向上のため、ハッシュタグ履歴等の一部データをブラウザのローカルストレージに保存する場合があります。

## 2. 利用目的
取得または保存される情報は、以下の目的で利用します。

- 入力条件に応じたプロンプトの生成
- ユーザー体験の向上（入力補助、タグ再利用など）
- サービス品質の維持・改善

## 3. 第三者提供
本サービスは、法令に基づく場合を除き、個人情報を第三者へ提供しません。

## 4. ユーザーの選択
ユーザーはブラウザ設定等により、ローカルストレージに保存されたデータを削除できます。  
削除後は、過去タグなどの保存情報は復元できません。

## 5. 個人情報の管理
本サービスは、氏名・住所・電話番号等の直接的な個人識別情報の取得を目的としていません。  
ただし、ユーザーが入力欄に個人情報を入力した場合、当該内容はユーザー自身の責任で取り扱うものとします。

## 6. 対象年齢
本サービスの利用に年齢制限は設けていませんが、未成年者は保護者の同意を得たうえで利用してください。

本ポリシーは、法令改正やサービス内容の変更に応じて改定される場合があります。改定後のポリシーは、公開時点から効力を生じます。
