# プライバシーポリシー

RichSolitaire（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、以下の通り情報の取り扱いを定めます。

## 1. 取得する情報
本アプリでは、以下の情報を自動的に取得する場合があります。

- 広告識別子（Advertising ID）
- 利用状況データ（クラッシュログ、使用時間など）

## 2. 利用目的
取得した情報は以下の目的で利用します。

- 広告配信（Google AdMob）
- アプリの改善
- 不具合解析

## 3. 第三者提供
本アプリは広告配信のために以下の第三者サービスを使用しています。

- Google AdMob

これらの事業者は、ユーザーの情報を収集し、広告配信やサービス改善のために利用する場合があります。
Googleによるデータ利用については以下をご確認ください。
https://policies.google.com/technologies/ads

## 4. ユーザーの選択
ユーザーは端末の設定により、広告識別子のリセットやパーソナライズ広告の制限を行うことができます。

## 5. 個人情報の管理
本アプリは、個人を特定できる情報（氏名、住所、メールアドレス等）を収集しません。
取得した情報は利用目的の範囲内で必要な期間保持されます。

## 6. 対象年齢
本アプリは13歳未満の児童を対象としていません。
